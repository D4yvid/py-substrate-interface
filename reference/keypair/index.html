    Keypair

    KeypairType ¶

    Type of cryptography, used in Keypair instance to encrypt and sign data

    • ED25519 = 0
    • SR25519 = 1
    • ECDSA = 2

    MnemonicLanguageCode ¶

    Available language codes to generate mnemonics

    • ENGLISH = 'en'
    • CHINESE_SIMPLIFIED = 'zh-hans'
    • CHINESE_TRADITIONAL = 'zh-hant'
    • FRENCH = 'fr'
    • ITALIAN = 'it'
    • JAPANESE = 'ja'
    • KOREAN = 'ko'
    • SPANISH = 'es'
